Computer Science Projects

Scratch Project

The Scratch Project was our first project in CSP. We were learning how to do pair programming to make a game in scratch. From scratch. My partner for this project was Surya Kethineni. The game that we made was a story mode game about a robber. We made the game interactive and fun, using cool dialogue lines and characters.

MIT App Inventor Project

This Project was about making an app in MIT app inventor. My partner and I based our game on Mole Mash V2. What we had to do was use an already made base project and give it cool features so that people would enjoy playing the game. We added features, such as, a point system, fire, a timer, and a high score.
